Ou siga estes 5 passos rápidos:

## 📱 COMO USAR O APP

### Tela 1: NOTAS
- Veja todas as suas notas fiscais
- Busque por valor, data ou descrição
- Toque em uma nota para ver detalhes completos
- Delete notas se necessário

### Tela 2: ADICIONAR
**Opção A: Scanner de QR Code**
1. Toque em "Iniciar Scanner"
2. Aponte para o QR Code da NF-e
3. O valor será detectado automaticamente
4. Complete os dados e salve

**Opção B: Inserir Manualmente**
1. Digite o valor
2. Selecione a data
3. Adicione descrição (opcional)
4. Escolha a categoria
5. Tire foto da nota (opcional)
6. Salve!

### Tela 3: RELATÓRIO
- Veja estatísticas gerais (total, média, mês atual)
- Filtre por data e categoria
- Gere relatório detalhado
- Compartilhe por email
- Exporte todos os dados (backup)

### ❓ Problemas Comuns

**Não vejo "Adicionar à Tela de Início"**
- Use o Safari (não Chrome)
- Toque no ícone correto: ⬆️
- Role o menu para baixo

**Câmera não funciona**
- Ajustes → Safari → Câmera → Permitir

**QR Code não lê**
- Use "Inserir Manualmente"

**Botão de ajuda no app**
- Toque no ícone **❓** no topo do app
- Abre o guia de instalação
